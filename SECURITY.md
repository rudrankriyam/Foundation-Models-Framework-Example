# Security Policy

If you believe you have found a security issue in `afm`, please do not open a public issue.

Instead, report it privately to:

- `rudrankriyam@gmail.com`

Please include:

- a clear description of the issue
- reproduction steps if available
- affected versions or commit hashes
- any suggested mitigation or patch context

I will review reports as quickly as possible and coordinate a fix before public disclosure when appropriate.
